# 🎓 Campus Placement Prediction using Machine Learning

This project predicts student placement status using machine learning algorithms based on academic and profile attributes.

## 📌 Project Overview
The objective of this project is to predict whether a student will get placed using academic performance, specialization, and work experience data.

Multiple machine learning models were implemented and compared:
- Random Forest
- Logistic Regression
- Decision Tree
- Support Vector Machine (SVM)

## 📊 Dataset
Source: Kaggle - Factors Affecting Campus Placement

Target Variable:
- Status (Placed / Not Placed)

Key Features:
- SSC Percentage
- HSC Percentage
- Degree Percentage
- MBA Percentage
- Work Experience
- Specialization
- Gender

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## 🚀 Model Performance

| Algorithm | Accuracy |
|------------|----------|
| Random Forest | 79% |
| Logistic Regression | 84% |
| Decision Tree | 81% |
| SVM | 58% |

Random Forest selected as final model due to balanced performance.

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- AUC Score
- Confusion Matrix
